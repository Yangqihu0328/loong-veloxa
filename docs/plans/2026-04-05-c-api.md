## Phase 1：C API 头文件 + 实现 + CMake

### 1.1 创建 `veloxa/api/veloxa_api.h`

C99 兼容公共头文件，包含：
- 不透明句柄类型定义
- VxResult 错误码枚举
- VxEventType / VxInputEvent / VxViewConfig 结构体
- 所有 C 函数声明

### 1.2 创建 `veloxa/api/veloxa_api.cc`

C++ 实现，每个 C 函数：
- NULL 参数检查 → 返回 VX_ERROR_NULL_PARAM
- `reinterpret_cast` 不透明指针 → C++ 对象
- 调用 C++ 方法
- 返回 VX_OK

关键映射：
- `VxView` → `vx::Application`
- `VxEventLoop` → `vx::platform::HeadlessEventLoop`
- `VxSurface` → `vx::platform::MemorySurface`
- `VxInputEvent` → `vx::event::InputEvent`

### 1.3 创建 `veloxa/api/CMakeLists.txt`

```cmake
add_library(vx_api STATIC veloxa_api.cc)
target_include_directories(vx_api PUBLIC ${PROJECT_SOURCE_DIR})
target_link_libraries(vx_api PUBLIC vx_core vx_graphics vx_platform)
```

### 1.4 修改 `CMakeLists.txt`

在 `add_subdirectory(veloxa/core)` 后添加 `add_subdirectory(veloxa/api)`。
